# QA-on-CLI-Nodejs
Short and Simple Question-Answer game on command line interface using nodejs

## QA_simple
This file includes implementation of QA game on CLI using process.stdin and process.stdout without any help from external module.

## QA
This file includes implementation of QA game on CLI using readline module and custom module. It uses collectAnswers.js file as custom external module.

## collectAnswers
This file is a module mainly created for the purpose of achieving separation of concerns and uses readline module in a function to ask questions to user and collect their answers.
